当鼠标悬停在元素上时，过渡的时间小于设置的时间；

当鼠标离开时，要停顿一下，元素的高度才开始减小

Hover Me!

HTML


<div class="demo">
    <h3><strong>Hover Me!</strong><h3>
</div>

CSS

            
.demo {
    width: 200px;
    height: 100px;
    max-height: 50px;
    overflow: hidden;
    background-color: #0aa;
    -webkit-transition: all .3s ease-in;
    transition: all .3s ease-in;
}
.demo:hover { 
    max-height: 500px;
}
